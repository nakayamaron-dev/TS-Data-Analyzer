# TS-DATA-ANALYZER

## Prerequisites

- Mac OS
  - Node.js installed (v16.14.2)

## How to set up

### initialize databases

First, build a database and write sample data.

```
bash init_db.sh
```

### Install Modules

```
npm install
```

### Build

```
npm run build
```

### Launch Web Server

```
npm run start-server
```

### Launch Angular

```
npm run start-front
```

### Launch Web Server by Docker

```
docker-compose up -d
```
